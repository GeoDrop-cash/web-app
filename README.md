# GeoDrop.cash
This is the front end code that controls [GeoDrop.Cash](https://geodrop.cash).

## Install
```
git clone https://github.com/GeoDrop-cash/web-app-geodrop
cd web-app-geodrop
npm install
```

## Usage
Development:
```
npm start
```

Production:
```
npm run build
```
